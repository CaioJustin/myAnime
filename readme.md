# MyAnimes

## Descrição do Projeto


Uma aplicação utilizando html,css,bootstrap e javascript que consome a api do [JIKAN](https://jikan.moe/) que fornece dados de forma não oficial do MyAnimeList

Atraves da manipulação do Dom, essa aplicação criar automaticamente cards no html.


## Feramentas Utilizadas

-HTML<br> 
-CSS<br>
-JAVASCRIPT<br>
-CSS<br>
-Bootstrap<br>
-API JIKAN<br>
